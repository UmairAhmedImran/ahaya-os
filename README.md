# ahaya-os

## Getting Started

Install [pnpm](https://pnpm.io) if you haven't already:

```bash
npm install -g pnpm
```

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Task 1 is covered in / page.
Task 2 is covered in /users page.
Task 3 is covered in /analytics page.
